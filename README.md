# Flipkart Sales & Performance Dashboard

This project presents an interactive dashboard analyzing Flipkart product data to uncover key insights related to sales performance, pricing strategy, customer ratings, and seller distribution.

## Key Features
-Analyzed product categories, brands, pricing, and discounts
-Identified top-selling categories and high-performing brands
-Evaluated customer satisfaction using ratings
-Studied pricing trends and discount impact on revenue
-Visualized seller distribution across different cities

## Dashboard Insights
-Toys and Electronics categories contribute the highest sales volume
-Sports brands like Nike and Adidas dominate product sales
-Most products fall within the mid-price range (₹10K–₹40K)
-Discounts play a key role in increasing sales performance
-Majority of sellers are concentrated in metro cities

## Tools & Technologies
-Power BI (Data Cleaning, Data Modeling, Visualization)
-DAX (Calculated Measures)
-Data Analysis Techniques

## Dataset
Flipkart product dataset (includes price, category, brand, rating, discount, seller details)

## Conclusion
This dashboard helps understand customer buying behavior, product performance, and pricing strategies, enabling better business decision-making.
